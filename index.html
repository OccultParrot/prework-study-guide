Prework Study Guide

HTML

  • The "head" element contains information about the webpage
  • The "body" element represents the visable content shown
  • Elements are opened and closed via "tags" surrounded by angle brackets
  • HTML is the foundation to any website

CSS

  • CSS makes websites prettier
  • You must link the CSS file in the HTML file using the element "link" with the arguments "rel" as stylesheet, and "href" pointing towards the CSS file
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside of an element

GIT

  • git status: Checks what branch we are currently working on
  • git checkout [branch-name]: Switches the branch we are on to the specified one
  • git checkout -b [branch-name]: Creates a NEW branch and switches the branch we are on to it
  • git add -A: Pushes all changed & added files to the staging area of git (-A means all files)
  • git commit -m "[short discriptive title here]": Pushes all files in the staging area of git to the working branch (-m means there is a message attached)
